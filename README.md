# 自习室座位预约系统(数据库加VX获取☟)

#### 介绍
自习室(阅览室)座位预约系统（SSM）LibrarySeatSystem
有BUG可留言加微
> VX：13033494971

#### 软件架构
Java + SSM（Spring+SpringMVC+Mybatis） + Mysql

#### 项目说明

1.  管理员功能模块
> - 角色管理
> - 用户管理
> - 教师管理
> - 学生管理
> - 班级管理
> - 添加阅览室
> - 选座，座位管理
> - 公告管理
> - 图书推荐
> - 数据统计

2.  用户功能模块
> - 前台首页
> - 开放时间信息
> - 座位预约
> - 个人信息
> - 读者推荐
